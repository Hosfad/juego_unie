# Practica 1

- Este es un juego de plataformas, El juego tiene una temática inspirada en `Palestina`, donde el objetivo principal del jugador es atravesar los diferentes obstáculos del nivel hasta llegar a la bandera palestina.

---

## Controles

El personaje puede moverse utilizando:

- **WASD**
    - `W` → Mover hacia delante
    - `A` → Mover a la izquierda
    - `S` → Mover hacia atrás
    - `D` → Mover a la derecha
    - `Space` → Saltar

- **Flechas del teclado**
    - `↑` → Mover hacia delante
    - `←` → Mover a la izquierda
    - `↓` → Mover hacia atrás
    - `→` → Mover a la derecha
    - `Space` → Saltar
